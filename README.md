# Berce E-commerce Store

A full-stack **E-commerce Web Application** built using the **MERN Stack with Next.js**. The platform provides a complete shopping experience including authentication, product browsing, cart management, wishlist, order placement, and an admin panel for managing the store.

---

## Live Demo

Frontend:  
https://berce-store.vercel.app  

Backend API:  
https://berce-backend.vercel.app

---

## Features

### User Features
- User registration and secure login using **JWT authentication**
- Browse products by **categories**
- Add products to **cart and wishlist**
- Place and manage **orders**
- Write **product reviews**
- Manage multiple **shipping addresses**
- Apply **discount coupons**
- View **order history**

### Admin Features
- Add, update, and delete **products**
- Manage **product categories**
- Create and manage **discount coupons**
- View and manage **orders**
- Monitor product reviews

## Architecture Overview

The application follows a **full-stack MERN architecture**:

Client (Next.js)  
⬇  
REST API (Express.js)  
⬇  
Database (MongoDB)

The frontend communicates with backend APIs which interact with MongoDB for storing and retrieving data.
